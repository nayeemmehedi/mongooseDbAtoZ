### Query and Projection Operators

## Comparison  1st 

        eq ->  value equal or not

        ne -> value not equal

        in -> array r vtr value ase ki na 

        nin -> array r vtr value nai na ki 
  
  
  ## Comparison 2nd 
  
  
        gt - greater then

        gte - greater then equal

        lt - less then

        lte - less then equal
  
  
  ## Logical
  
        and 

        or

        not
